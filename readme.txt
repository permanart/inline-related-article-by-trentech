Terima kasih telah memasang plugin <a href="https://www.trentech.id">Inline Related Article by Trentech.id</a><
Cara menggunakan plugin ini sangat <em>simple,</em> silahkan <em>copy shortcode</em> berikut :

[post jumlah=a tag="b"]

a = Jumlah Artikel, kami buat maksimal 5 
b = tag artikel yang diinginkan, apabila ingin lebih dari satu tag maka tambahkan plus <strong>+</strong>, contoh <strong>[post jumlah=3 tag="startup+bisnis"]</strong>. Apabila tag kamu lebih dari satu kata, misalkan kamu memiliki tag <em>"belajar koding"</em> maka diantara kedua kata tersebut tambahkan tanda <strong>-</strong>, contoh <strong>[post jumlah=3 tag="belajar-koding"].</strong> </br></br>

Terima kasih telah menggunakan plugin ini, saran dan masukan sangat berharga untuk kami.
